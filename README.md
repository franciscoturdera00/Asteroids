
# Asteroids

A recreation of the popular game [Asteroids](https://en.wikipedia.org/wiki/Asteroids_(video_game)) in `python3`


## Prerequisites:
```
python 3.13.1 or later
pygame 2.6.1 or later
```

### On ~/.../Asteroids:

#### Install pygame
```
pip install pygame
```

#### Run Game
```
python main.py [--p2]
```
--p2: Two Player
